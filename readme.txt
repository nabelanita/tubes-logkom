Spesifikasi lingkungan
- Program harus dijalankan di terminal linux/mac OS/windows subsystem for linux.
- Program bisa saja dijalankan di GNU prolog console, tetapi tidak disarankan karena 
  tidak mendukung untuk menampilkan efek dari game.

Cara mengeksekusi program
0. Untuk Windows, install Windows Subsystem for Linux terlebih dahulu.
   Untuk mac OS, install homebrew terlebih dahulu.
1. Install gprolog dengan menjalankan command: 'sudo apt-get install gprolog' di terminal.
   Untuk mac OS, install gprolog dengan menjalankan command: 'brew install gnu-prolog' di terminal.
2. Buka terminal pada directory yang sesuai.
3. Ketik 'gprolog' pada terminal.
4. Masukkan ['main.pl']. pada query.
5. Setelah main.pl berhasil dicompile, ketik 'openGame.' pada query untuk memulai game.
6. Game berhasil dijalankan
